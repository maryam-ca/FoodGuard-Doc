process

How FoodGuard works

From image to freshness category + live AI advice.

01
📸

Upload or capture

Take a photo of any food item — fruit, vegetable, leftovers. The UI streams it to backend.

02
🧠

Vision AI analysis

Python backend preprocesses (resize, denoise) and runs model to detect mold, discoloration, texture.

03
💬

Gemini live agent

You can ask “why is it spoiled?” or “how to store?” – Gemini answers conversationally.

capabilities

Core features

🍞

Mold & decay detection

Trained model spots visual indicators: green spots, texture breakdown, color fading.

🥑

3‑stage freshness

Fresh · Moderate risk · Spoiled. With clear recommendation: eat, cautious, discard.

🤖

Gemini live Q&A

Ask anything about the food, storage tips, or spoilage signs — instant AI answer.

⚙️

FastAPI + React

Scalable backend, modern frontend. Easy to deploy on cloud or local.
